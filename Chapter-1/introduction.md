# Chapter 1: Introduction

## Background

Cloud computing allows businesses to use internet-based resources instead of physical infrastructure.

## Problem Statement

Traditional systems are costly, not scalable, and difficult to maintain.

## Objectives

* Improve efficiency
* Reduce cost
* Enhance accessibility
* Ensure security

## Scope

Applicable in banking, healthcare, e-commerce, and education.

## Existing System

On-premise systems with high cost and low flexibility.

## Proposed System

Cloud-based system using AWS services.

## Technologies Used

* AWS
* Virtualization
* Web technologies
